## функционал

1. Показать случайное английское слово


 2. Проверить свой перевод

Вы вводите перевод показанного слова, а программа говорит правильно или нет.

 3. Добавлять свои слова

Можно ввести английское слово и его перевод и оно добавится в общий список.

 4. Смотреть весь список слов

Приложение показывает все слова, которые есть в хранилище.

 5. Искать слова

Можно найти слово по части английского слова или по части перевода.

 6. Получать слово по его ID

Если указать ID слова приложение покажет полную информацию о нём.

 7. Изменять слово

Можно изменить либо само английское слово, либо его перевод.

 8. Удалять слово

Можно удалить слово из списка по его ID.
